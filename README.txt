# Sistema de Autenticación (Login) - Java Swing 

Aplicación de escritorio que implementa un flujo completo de autenticación de usuarios, registro y visualización de perfil.

## Características

* **Inicio de Sesión:** Validación de credenciales contra una lista en memoria.
* **Registro de Usuarios:** Formulario para crear nuevos usuarios con roles definidos.
* **Perfil de Usuario:** Visualización dinámica de los datos del usuario logueado.
* **Arquitectura:**
    * Separación de Lógica (`UserManager`, `Usuario`) y Vista (`JFrame`).
    * Uso de una librería de utilidades personalizada (`SwingUtils`) para manejo de GUI.

## Tecnologías

* Java (JDK 21)
* Java Swing (Interfaces Gráficas)
* NetBeans IDE
